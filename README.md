## Como publicar no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload dos arquivos (`index.html`, `README.md`)
3. Vá em Settings > Pages
4. Em “Source”, escolha a branch `main` e pasta `/ (root)`
5. Clique em Save
